CMDs - definition:
--browserClient=chrome:		not required (defaults to chrome if not declared)
--browserStack=false:		not required (defaults to *.config.js file's settings if not declared)

- TO RUN THE TESTS DO ONE OF NEXT WITHIN YOUR CMDER:
:::: run tests locally (mainly for debugging purpose) ::::
npm run uclaw:test1 -- --env=tst --appName=ucLaw --browserStack=false
npm run uclaw:test1 -- --env=tst --appName=ucLaw --browserClient=chrome --browserStack=false

:::: when running the test in browserStack run one of next commands ::::
npm run uclaw:test1 -- --env=tst --appName=ucLaw --browserStack=true
npm run uclaw:test1 -- --env=tst --appName=ucLaw --browserStack=true --browserStackArgs=Windows,7,Chrome,69.0,1024x768
